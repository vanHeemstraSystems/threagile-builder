Welcome to Threagile Builder's documentation!
===================================

**Threagile Builder** (/thre'gile/) is an Threagile Threat Model Editor for the creation of threagile yaml files
that contain threat models.
See threagile at `threagile.io <https://threagile.io/>`_
and offers a *visual* and *low-code* editing environment.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
